# Design Component
## Bars
**Types:**
1. Fixed -default
2. Slider: show it's handle
3. Floater: float on page

**Sizes:**
1. Normal -default
2. Larger
3. Full

## Boxes
**Types:**
1. Untitled -default
2. Titled

## Buttons/Links Group
**Types:**
1. Titled -default
2. Icons: just show button icon
